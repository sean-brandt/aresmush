The rename command lets you change an object's name.  You do not need to be in the same room as the object unless there are multiple objects with the same name.

    %xcrename <name or database ID>=<new name>%xn - New name.
    
    The examine command lets you see the contents of the database for a given object.  You do not need to be in the same room as the object unless there are multiple objects with the same name.

        %xcexamine <name or database ID>%xn - Examines an object.
